But then I must include the generated files in source control and fix any newly added files.\nThere doesn't seem to be any settings to change the encoding.\nIs this a bug in the compiler?\nThe project type is jsproj, which doesn't seem to support tsconfig.json according to https://github.com/Microsoft/TypeScript/issues/3124\n\nA:\n\nOnce I manually fix encoding of the generated files it's fine.
