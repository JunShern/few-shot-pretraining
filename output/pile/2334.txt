Everything seems to work whether ido-everywhere is set or not.\n\nA:\n\nido-everywhere function\n(define-minor-mode ido-everywhere\n  \"Toggle use of Ido for all buffer/file reading.\nWith a prefix argument ARG, enable this feature if ARG is\npositive, and disable it otherwise.  If called from Lisp,\nenable the mode if ARG is omitted or nil.\"\n  :global t\n  :group 'ido\n  (remove-function read-file-name-function #'ido-read-file-name)\n  (remove-function read-buffer-function #'ido-read-buffer)\n  (when ido-everywhere\n    (add-function :override read-file-name-function #'ido-read-file-name)\n(add-function :override read-buffer-function #'ido-read-buffer)))\n\nIt overrides read-file-name-function (https://www.gnu.org/software/emacs/manual/html_node/elisp/Reading-File-Names.html), read-buffer-function (ftp://ftp.gnu.org/old-gnu/Manuals/elisp-manual-20-2.5/html_chapter/elisp_20.html).\nYou can see the effect when you try File->Open File from menu bar.\nWith ido-everywhere disabled, it opens graphical interface, but with ido-everywhere enabled it shows file list in mini buffer (ido style).\nEffect can be seen where ever these overrided functions are used.\n\n"
